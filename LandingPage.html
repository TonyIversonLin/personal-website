Sage

This is personal financial managent app that can keep track on your transaction, budget, financial goal. The unique feature this app is that client can download the whole app and install it on their personal Raspberry Pi. It means that client is able to host their own server that offer free basic financial management service without worrying the third party(or middleman) stealing personal financial data and interruption of embedded advertisements. In addition, the app can auto-categorize your future transaction based on client's existing transaction by using smart matching algorithm. All the personal data is safely protected because the app is guarded by very reliable encryption strategy. Even if the Raspberry Pi is stolen, client banking information won't be stolen since those data will be encrypted by client password in the first place.


Technology Used

ReactJS/Redux, Banking.js, Victory.js, React-Bootstrap, Node.js, Express.js, Sequelize, PostgreSQL, RESTful API server, AJAX
